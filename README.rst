wechat-platform
=========================

微信公众号自助开源平台

当前进度：正在开发中，具体进度请查看 develop 分支

简要概述
-------------------------

想法很简单：

* 对运营者，一个5分钟可以立即使用的包含基本功能的微信公众号平台，更多扩展功能以插件的形式，即插即用

* 对开发者，一套完善的插件机制，提供完整的API接口，专注业务逻辑处理，同时可将插件打包发布，供他人使用

将在发布第一个版本后接受Pull Request，欢迎大家一起完善这个平台。
